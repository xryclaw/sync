# SLS 日志分析平台

SLS 日志分析平台。构建一个 Web 平台，通过上传 CSV 日志文件，自动解析玩家行为链路，可视化展示操作时间线，并识别 bug 触发前的异常模式。

## 功能列表

*   **项目初始化**: 初始化项目环境，包括目录结构、配置文件等。
*   **后端基础架构**: 搭建 Node.js + Express 后端服务，配置 Multer 和 csv-parser。
*   **CSV 文件上传功能**: 实现 CSV 文件的上传、解析和数据存储到 SQLite 数据库。
*   **玩家行为分析 API**: 提供用户行为时间线、Error 上下文和行为统计等 API 接口。
*   **前端项目初始化**: 搭建 React + Vite 前端项目，配置 Ant Design、ECharts、Axios。
*   **文件上传页面**: 开发前端文件上传页面，支持拖拽上传和历史会话展示。
*   **时间线可视化组件**: 使用 ECharts 绘制时间线图，支持缩放、筛选、高亮 error 事件。
*   **异常分析页面**: 开发异常分析页面，展示 error 前的操作序列和行为对比。
*   **Docker 部署配置**: 编写前端和后端 Dockerfile，配置 docker-compose.yml 实现容器化部署。
*   **测试与优化**: 进行单元测试、性能优化和错误处理完善。
