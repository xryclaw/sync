## 🎯 MVP 功能范围

**核心流程：**
```
上传 CSV → 解析存储 → 表格展示 → 筛选（时间/级别/关键字）→ 查看详情
```

**技术栈：**
- 后端：Node.js + Express + SQLite
- 前端：React + Vite + Ant Design

**开发时间：** 10-15 小时
